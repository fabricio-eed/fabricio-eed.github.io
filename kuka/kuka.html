Offset a Target Point:



the Geometric operator applies shifts (the portion on the right side of the colon) along/around the reference frame (the portion on the left side of the colon).
PTP P1 : {X 0,Y 0,Z 100,A 0,B 0,C 0} will move the robot to a point 100mm from P1, along the Z+ axis of the active $TOOL,
PTP {X 0,Y 0,Z 100,A 0,B 0,C 0} : P1 will move the robot to a point 100mm from P1, along the Z+ axis of the active $BASE.

Lin_Rel and Variables:



1. use aggregate (curly braces). this un-initializes variable and then initializes only mentioned elements. assigned values must be literals. example: 


DECL POS myPoint
; ...
MyPoint = { x 100} ; x element is set, all others are unknown
MyPoint = { y 200} ; y element is set, all others are unknown - including X which used to be 100
MyPoint = { x 100, y 200, z 300, a 0, b 45, c 60, s 6, t 21} ; can only use constants such as 100, 200, 300, 6, 21 but not variable names

2.2. use dot operator. this allows you to access single element WITHOUT affecting (loosing) values of other elements. this also allows you to use variables, expressions etc. 


DECL POS myPoint
; ...
MyPoint = { x 100, y 200, z 300, a 0, b 45, c 60, s 6, t 21} ; can only use constants such as 100, 200, 300, 6, 21 but not variable names
MyPoint.y= MyPoint.y - 3*$date.sec + sin(45) ; we can use expressions, variables etc. only .Y component is affected, all other elements of MyPoint are retained
